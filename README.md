# Islam Eltahan – Bilingual Portfolio (GitHub Pages)

This is a fast, responsive, bilingual (EN/AR) portfolio website.

## Deploy on GitHub Pages
1. Create a new repository (e.g. `islam-eltahan-portfolio`).
2. Upload all files in this folder to the repo root.
3. GitHub → **Settings** → **Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
4. Your site will be available on the GitHub Pages URL.
